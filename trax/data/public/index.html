Getting started

Here’s how to get rolling:

  1. Create your databases and edit config/database.ini

    Trax needs to know your login and password.

  2. Use script/generate.php to create your models and controllers

    To see all available options, run it without parameters.

  3. Set up a default route and remove or rename this file

    Routes are setup in config/routes.php.
